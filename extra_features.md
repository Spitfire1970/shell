# History

Allows user to browse through previously entered commands by pressing up and down keys

# Extra Applications

## mkdir

Creates a new directory in your current working directory

## touch

Creates a new file in your current working directory

# Custom Exceptions

## InvalidRangeError

Throws error in cut if invalid range

## NoDirectoryFoundError

Throws error if given directory does not exist

## NoFileFoundError

Throws error if given file does not exist

## UnsupportedApplicationError

Throws error if given application does not exist

## WrongFlagsError

Throws error if given flag does not exist

## WrongNumberOfArgsError

Throws error if wrong number of arguments are given to an application

## WrongNumberOfRedirectionsError

Throws error if wrong number of redirections are given
